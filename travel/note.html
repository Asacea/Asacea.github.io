# 一些随想

# 我从很小的时候就和爸爸妈妈一起开着车去旅行。

四川是盆地，有很多山。我的父母相信，相比于城市的繁华和喧嚣，川西的宁静广阔更适合我的成长。而我，也总是期待着爸爸妈妈去学校帮我请周五下午的假，好带上我一起在川西度过快乐的旅行周末。

在川西，小城和小城之间要穿过很多山和草原，有时我们为了赶到目的地，要开一整天的车。那时候，山路上时常没有网，我也没有手机，甚至车载收音机也是断断续续。爸爸妈妈在前排交流着路况，我独自一人坐在汽车的后排，扭过头注视着窗外的风景，看着太阳慢慢地落到蜿蜒的河流中，像是草原上的赤色飘带。我们就这样顺着小河，追着太阳的余晖，向草原深处开去。

有一天晚上，我们在露营点露营。天冷极了，我们和陌生人一起升起了篝火，热一些水，泡几壶茶。漫天的星星好像在不停地旋转，我用我贫瘠的天文常识，竭力从中辨认几个星座，终究是失败了。我好像能感受到那些星星，感受到它们像河流一样流动，忽远忽近，彼此照应。更多的记忆却消失，或许那时候我太小了，只隐约记得那个晚上睡得心惊胆战，附近牧民养的牛没有系绳子，睡觉时总能听到它们在我们的帐篷边嗤嗤的声音。。。。。。 这些旅行记忆弥足珍贵，我却在一点一点的遗忘。

# 现在我长大了，出去旅行的时间却少了。

回想到，自成年以来，我好像再也没有和爸爸妈妈一起出去旅行过了。都说，脱离父母的怀抱是成长的必经之路，有时我却不免感到遗憾。

想来这些年，我更希望和我的同龄人一起旅行。可是大家的时间却很难凑到一起，想法也是天马行空。一起制定了大大小小的成百上千个计划，完成的、能称得上旅行的，一只手都能数得过来。旅行得过程总是充满摩擦、争吵和相互忍让，总的来说，还是快乐比伤心多，满足比遗憾多，惊喜比惊吓多。在旅行的过程中，我总是能学到很多，感悟到很多。

# 总而言之，

我决定把我的旅行记录下来，有什么感想感悟都留下痕迹。作为一个enfp，想到就要做到！！！
