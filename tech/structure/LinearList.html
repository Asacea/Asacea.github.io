# 逻辑结构

  1. 特点：
    • 同一性：由同类元素组成
    • 有穷性：由有限个元素组成
    • 有序性：相邻元素之间存在序偶关系

# 顺序结构

  1. 顺序储存结构

    这里的顺序表其实就是数组

    #define MAX_SIZE 100
typedef struct{
    ElemType elem[MAX_SIZE];
    int last;    //记录线性表中最后一个元素在elem数组 当中的位置，空表置为-1，因此可以得出，线性表当中元素的实际个数=last+1，即顺序表的长度；
}SeqList;

    ai在线性表当中的序号为i，在顺序表对应的数组elem中的下标为i-1

  2. 定义变量的两种方法：

    • SeqList类型的变量：SeqList L;。访问i号元素：L.elem[i-1];最后一个元素的下标:L.last;顺序表的长度：L.last+1
    • 指针变量：SeqList L1,*L; L=&L1 。访问i号元素：L->elem[i-1];顺序表的长度：L->last+1

  3. 基本运算

    • 查找操作：

      1. 按序号查找：

        int GetData(SeqList L,int i){
    if(i<=L.last)
        return L.elem[i-1];
    else
        return 0;
}

      2. 按内容查找：

        int Locate(SeqList L,ElemType e)
{
    int i=0;
    while((i<=L.last)&&(L.elem[i]!=e))
        i++;
    if(i<=L.last)
        return i+1;  //返回其序号
    else
        return -1;   //返回空置符-1
}

        该算法的时间复杂度为O(n)

    • 插入操作：

      其实就是插入数组操作,先找到插入位置，然后将该位置后的元素都依次向后移动

      #define OK 1
#define ERROR 0
int InsList(SeqList *L,int i,ElemType e){
    int k;
    if((i<1)||(i>L->last+2))
        return ERROR;
    if(L->last>=MAX_SIZA-1){
        printf("no space");
        return ERROR;
    }
    for(k=L->last;k>=i-1;k--)//倒序
        L->elem[k+1]=L->elem[k];
    L->elem[i-1]=e;
    L->last++;
    return OK;
}

      算法分析：E为在长度为n的表中插入一个元素所需移动元素的平均次数。假设插入 任何位置的可能性相当，E=n/2

    • 删除操作：

      将目标位置的所有元素一次向前依次移动一位

      int DelList(SeqList *L,int i,ElemType *e){
    int k;
    if((i<1)||(i>L->last+2))
        return ERROR;
    *e=L->elem[i-1];
    for(k=i;k<=L->last;k++)
        L->elem[k-1]=L->elem[k];
     L->last--;
    return OK;
}

      E=(n-1)/2

    • 合并运算：

      将两个顺序表合并成一个顺序表，所有顺序表元素都是非递减有序排列（即单调不减）

      void mergeList(SeqList *LA,SeqList *LB,SeqList *LC){
    int i,j,k;
    i=0;
    j=0;
    k=0;
    while(i<=LA->last&&j<=LB->last){
        if(LA->elem[i]<=LB->elem[j]){
            LC->elem[k]=LA->elem[i];
            k++,i++;
        }else{
            LC->elem[k]=LB->elem[j];
            k++,j++;
        }
    }
    //当某一表当中的元素有剩余的时候，将剩下的元素直接赋给LC
    while (i<=LA->last){
        LC->elem[k]=LA->elem[i];
        i++,k++;
    }
    while (j<=LB->last){
        LC->elem[k]=LA->elem[j];
        j++,k++;
    }    
}

# 循环链表

  1. 定义方法：将单链表的最后一个结点的指针域，由指向NULL改为指向头结点

  2. 判断当前结点p是否是尾结点p!=L或p->next!=L

  3. 分类：带头结点和带尾指针rear

  4. 初始化(带头结点)

  5. 建立循环单链表:

    假设已经进行了初始化

    void CreateCLinkList(LinkList CL)//CL是头指针
{
    Node *rear,*s;
    char c;
    rear=CL;
    c=getchar();

        s=(Node*)mallco(sizeof(Node));
        s->data=c;

  6. 合并两个循环列表

    //LA表示头指针
LinkList merge_CLists(LinkList LA,LinkList LB){

}

  • 读取第一个数据data，后续比它大的放在该结点的前面，比它小的放在后面
  • 逻辑：比它小的结点头插，比它大的不管
  • 图解：见逆置
vod changelist(LinkList L){
    Node *a,*p,*q;
    if(L->next==NULL)
        return ;
    a=L->next;
    p=a->next;
    q=p->next;
    while(p!=NULL){
        if(p->data>=a->data){
            p=q;
            q=q->next;
        }else{
            p->next=L->next;
            L->next=p;
            p=q;
            q=q->next;
        }
    }
}

# 单链表二进制加一

  • 建表：第一个结点存储二进制数的最高位，依次存储，最后一个结点存储最低位。
  • 实现加1运算：从最低位往前，找到第一个0的位次所在结点，变成1；从该结点向后，所有的1都变成0，
void BinAdd(LinkList L){
    Node *p,*goal,*s;
    p=L->next;
    r=L;
    while(p!=NULL){
        if(p->data==0)   //找到最后一个为0的位置
            r=p;
        p=p->next;
    }
    if(r!=L)
        r->data=1;
    else{
        s=(Node*)mallco(sizeof(Node));  //申请新结点存放最高进位
        s->data=1;   //赋值为1
        s->next=L->next;    //将新结点头插入
        L->next=s;
        r=s;
    }
    r=r->next;
    while(r!=NULL){//将后面的所有结点赋值为0
        r->data=0;
        r=r->next;
    }
}
