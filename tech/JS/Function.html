# 函数 Function

理解Javascript中的函数：函数实际上是对象，每个函数都是Function类型的一个实例。

函数的声明：

function sum(num1,num2){
    return num1+num2;
}//函数声明
let sum = function(num1,num2){
    return num1+num2;
};//函数表达式
let sum = (num1,num2)=>{
    return num1+num2;
};//箭头表达式

# 理解函数

  1. 函数名可以理解成指向函数对象的一个指针。这一点可以通过函数的创建方式之函数表达式看出。因此，可以通过let func = sum;来指向函数内存空间，使用sum =null;来使sum这个函数名与函数断开联系。

  2. 函数参数：函数的参数可以通过arguments关键词访问，arguments.length获得参数的个数，arguments是数组，可以通过arguments[]修改对应形参。arguments不能反映参数的默认值。可以在函数内部使用arguments.callee()来调用当前函数。

    • 函数参数的默认值

      function ajax(url,method="get",async=true){
    console.log(...arguments);
}
ajax("/aaa","post");

  3. 不能重写，后定义的会覆盖前面的;

  4. 声明提升：函数声明会提升到上下文的顶端，但是通过函数表达式声明的函数不会。

# 箭头函数

  1. 箭头函数的写法：let func = (arg1,arg2..)=>{return ...}

  2. 箭头函数与标准函数的区别：

    1. arguments：箭头函数不能使用arguments\super\new，可以使用扩展符收集参数成数组

      let sum = (...values)=>{
    return values.reduce((x,y)=>x+y,0)
}

    2. this：标准函数中，this指向的是函数被调用时的上下文；在箭头函数中，this指向的是函数定义时的上下文，或者说”父作用域“

      标准函数绑定this的方法: apply(), call(), bind()

      //apply()和call()接收的第一个参数就是要指向的this，后面的参数则是函数需要的参数，区别是apply可以传入数组，call必须要将参数逐个传入
window.name="window"；
let obj = {
    name:"obj"
};
function sayName(){
    console.log(this.name);
}
sayName();	//window
sayName.call(obj);	//obj
let say=sayName.bind(obj);
say();	//obj

      箭头函数

      <input type = "text" id = "search">
<script>
let search = document.querySelector("#search");
search.oninput = function(){
        setTimeout(function(){
            console.log(this.value);//这里的this指向了调用时的上下文，setTimeout是在这个函数结束后才被window调用，所以会打印undefined。
        },1000);
    }
search.oninout = function(){
    setTimeout(()=>{
        console.log(this.value);//箭头函数的this始终指向定义函数时的上下文，即本身
    },1000);

</script>

# 闭包closure

  1. 闭包的含义：闭包的定义是那些引用了另一个函数作用域中变量的函数。

  2. 作用域链：在函数定义的时候就会创建作用域链，包含两部分，活动对象和变量对象。

    活动对象是指函数局部上下文中的对象（包含函数的arguments和具体的参数），在定义函数时创建函数的活动对象，只在函数执行期间存在；变量对象是全局上下文，在调用函数时创建。？？？？

  3. 举例

    function createComparison(property){
    return function(obj1,obj2){
        let val1=obj1[property];
        let val2 = obj2[property];
        if(val1<val2){
            return 1;
        }else if (val1==val2){
            return 0;
        }else{
            return -1;
        }
    };
  }
  let compare = createComparison("id");
  let result = compare({id:1},{id:2});
  console.log(result);

    在这个例子中，createComparison函数中返回了一个匿名函数，这个匿名函数在定义的时候，它的作用域链上包含了createComparison的全局变量对象（createComparison和result），createComparison的活动对象（property，arguments），以及它自己的活动对象（arguments，obj1，obj2）——闭包。createComparison调用结束后它的活动对象任然存在在这个匿名函数的作用域中。

  4. 闭包中的this

    let obj = {
    id:"id111",
    getId(){
        let that = this;//这里的this 指向obj,使用一个变量来保存
        return function(){
            return that.id;//这里如果用this，this指向调用函数时的上下文
        }
    }
}
