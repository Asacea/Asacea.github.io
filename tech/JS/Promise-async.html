# Promise & async，await

# Promise

  1. 基本使用

    let p = new Promise(function(resolve,reject){//执行函数的2个参数是2个函数
    setTimeout(()=>{
        resolve("res");
    },1000);
});
p.then((res)=>{
    console.log("resolve:",res);
},(arr)=>{
    console.log("reject",arr);
});

  2. Promise对象的状态

    Promise对象有3中状态：待定pending，兑现fulfilled，拒绝rejected。其中pending是初始状态，由pending状态转化为fulfilled或rejected状态是不可逆的过程。

    控制promise的状态转换时通过调用它的2个函数参数resolve和reject来实现的，调用resolve()把状态切换成fulfilled，调用reject把状态切换成rejected，还会抛出错误。

    例如下面这段代码，实现了一个定时退出功能。

    let p = new Promise((resolve,reject)=>{
        setTimeout(reject,5000);//设置成5000ms后回调reject
     console.log("doing other things");
      });
setTimeout(console.log,0,p);//这时的状态是pending
setTimeout(console.log,6000,p);//此时是rejected
/*
doing  //这里先打印doing是因为执行器函数是promise的初始化程序
Promise {<pending>}
Uncaught (in promise) undefined
Promise {<rejected>: undefined}
*/

  3. then

    .then()方法为promise添加onResolved和onRejected两个处理方法，分别进入fulfilled和rejected状态。.catch()方法相当于.then(null,onRejected)作为拒绝处理程序。.finally()最终一定会执行。

    注意：每个promise的实例方法（then\catch\finally）的返回值都是一个新的promise。

    尝试使用promise来封装ajax请求

    function ajax(url){
       return new Promise((resolve,reject)=>{
           let xhr = new XMLHttpRequest();
           xhr.open("get",url,true);
           xhr.onreadystatechange = function(){
               if(xhr.readyState === 4){
                   if(xhr.status >= 200&&xhr.status <300){
                       resolve(xhr.responseText);
                   }else{
                       reject(xhr.responseText);
                   }
               }
           }
       })
   }
   ajax("/aaa").then(res=>{
       console.log(res);
   }).catch(err=>{
       console.log(err);
   });

# 异步函数

什么是异步操作？

异步操作指的是程序执行过程中，某个任务或操作不会阻塞程序的继续执行，而是以非阻塞的方式进行。这允许程序在执行异步操作的同时继续执行后续代码，而不必等待异步操作完成。

# async

async关键词用于声明异步函数，让函数拥有异步特征。异步函数返回一个promise对象，如果没有return，返回undefined；如果return了一个非promise的对象，直接将状态切换成了fulfilled，相当于通过onResolved方法传递了这个值；如果返回了一个promise对象，那么在后续代码的执行中可以使用.then()等方法，可以解包，结合setTimeout等方法实现异步。例如：

async function test(){
    console.log("test");
    return new Promise((resolve,reject)=>{
        let r = Math.random()*10;
        if(r>5) resolve(r);
        else reject(r);
    })
}
test().then(res=>{
    console.log("resolve",res);
},err=>{
    console.log("reject",err);
});

# await

await关键词可以暂停异步函数代码的执行，让出Javascript运行 时的执行线程，等待promise解决。注意await只能出现在异步函数中，否则会抛出错误SyntaxError。await 与async配套使用，解决了异步操作的问题，还可以实现顺序的链式调用。

let count = 0;
async function sleep(delay){
    count++;
    console.log(`sleep ${count}`);
    return new Promise((resolve,reject)=>{
        setTimeout(()=>{
            console.log("sleeping",delay);
            resolve(delay+1000);
        },delay);
    });
}
async function test(){
    let res = await sleep(1000);
    console.log("interval");
    let res2 =await sleep(res);
    console.log("finish test");
}
test().then(res=>console.log(res));
setTimeout(console.log("finished"),0);//同步
//sleep 1-finished-sleeping 1000-interval-sleep 2-sleeping 2000-finish test-3000

如果await等到了一个reject的结果，那么直接走catch回调函数的语句执行。这种错误处理两种方式，一种是直接try-catch，另一种是在用promise的.catch()

使用async/await封装Ajax请求

function ajax(url){
   return new Promise((resolve,reject)=>{
       let xhr = new XMLHttpRequest();
       xhr.open("get",url,true);
       xhr.onreadystatechange = function(){
           if(xhr.readyState === 4){
               if(xhr.status >= 200&&xhr.status <300){
                   resolve(xhr.responseText);
               }else{
                   reject(xhr.responseText);
               }
           }
       }
   })
}
async function test(){
    let res =0;
    res = await ajax('1.json');
    console.log(res);
}
